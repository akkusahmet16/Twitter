# Old Twitter iOS App 🐦

A SwiftUI iOS app that recreates the classic blue bird Twitter UI.  
Goal: **clone the look & feel of old Twitter**, using static/mock data only — no real Twitter/X API integration.

> This project is purely for learning and nostalgia purposes. It is not affiliated with Twitter/X in any way.
